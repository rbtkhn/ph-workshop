# Registries

This folder holds public registries after the first manual Geo-Strategy batches proved the annotation shape.

The active typed registry is [cross-volume-links.yaml](cross-volume-links.yaml), which supports public guided corridors under [corpus/cross-volume/](../corpus/cross-volume/).

Possible registries:

- sources
- cross-volume links
- concepts
- claims
- predictions
- counter-readings
- reading paths

Prediction tracking should be core but scoped: prominent for Geo-Strategy and current-events claims, lighter for non-forecast corpora.
