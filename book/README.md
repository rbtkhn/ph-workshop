# Book

This folder holds public book and site material derived from the reviewed corpus.

The book surface should not outrun the source surface. Public book material should be added only when the underlying corpus items, annotations, and source statuses are clear enough for readers to verify.

The default reading unit is one chapter at a time: Part I is the transcript, Part II is the companion commentary. Use [chapter-manifest.yaml](../chapter-manifest.yaml) as the routing table when you want to open or compare a chapter pair.
