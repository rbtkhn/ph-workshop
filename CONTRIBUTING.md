# Contributing

Pull requests and issues are welcome, but this repository is source-disciplined. Contributions should help Jiang students understand the corpus more clearly without weakening attribution, uncertainty labels, or review status.

## Contribution Rules

- Preserve source IDs when known, such as `geo-01`, `vi-16`, or `es-13`.
- Include source URLs, publication dates, and timestamps for video-derived claims where practical.
- Separate source text, summary, interpretation, prediction, and disagreement.
- Do not add unsupported claims.
- Do not present representation as endorsement.
- Do not import private notes, internal workbench files, or unpublished working drafts.

## Rights And Source Text

By contributing, you confirm that you have the right to submit the material you add.

Do not paste large transcript blocks, full essays, paywalled material, or copyrighted source text unless the repository has explicitly accepted that source-text policy. Short excerpts may be proposed for study use, but longer excerpts and curated transcript text require rights/review before publication.

When in doubt, link to the original source and summarize in your own words.

## Corrections

Use issues or pull requests for:

- transcript corrections
- source metadata corrections
- changed or broken links
- annotation disagreements
- new source suggestions

See [docs/corrections-policy.md](docs/corrections-policy.md).

## Review Expectations

Maintainers may ask contributors to:

- add source timestamps
- reduce quotation length
- clarify whether a note is summary or interpretation
- add uncertainty/status labels
- move broad disagreement into a counter-reading note

The goal is not to make the repo timid. The goal is to make it useful, traceable, and fair.
