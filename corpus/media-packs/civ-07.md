# Civilization #7: Homer's Iliad and the Birth of Greek Civilization Media Pack

## How To Use This Pack

Use this pack before reading the transcript or commentary. It gives students a visual and documentary doorway into the lecture while keeping source claims, interpretation, and rights status separate. This is a curated draft awaiting human search and curation.

## Entry Object

### 01. Achilles and Ajax gaming amphora

- **Type:** artwork
- **Source:** https://commons.wikimedia.org/wiki/File:Exekias_Achilles_Ajax_Staatliche_Antikensammlungen_1471.jpg
- **Rights:** external_link_only
- **What to notice:** Greek vase painting gives the lecture a visual doorway into heroic competition, aristocratic honor, and the social memory carried by Homeric myth.
- **Limit:** A later painted object is not a direct image of Homeric society; it shows how Greeks remembered and re-staged heroic values.

## Context Anchors

### 02. Mycenae Lion Gate

- **Type:** place
- **Source:** https://commons.wikimedia.org/wiki/File:Lion_Gate,_Mycenae.jpg
- **Rights:** external_link_only
- **What to notice:** Mycenae anchors the Homeric world in the palace and warrior culture that later Greeks imagined as ancestral.
- **Limit:** The archaeological site cannot be read as a simple confirmation of the Iliad as history.

### 03. Map of Homeric Greece

- **Type:** map
- **Source:** https://en.wikipedia.org/wiki/Homeric_Greece
- **Rights:** external_link_only
- **What to notice:** A regional map helps students see Homer as a civilizational memory system spread across places, routes, and rival communities.
- **Limit:** Modern maps simplify uncertain Bronze Age and archaic geographies.

## Primary Objects And Texts

### 04. Linear B tablet

- **Type:** artifact
- **Source:** https://commons.wikimedia.org/wiki/File:NAMA_Linear_B_tablet_of_Pylos.jpg
- **Rights:** external_link_only
- **What to notice:** The tablet points to bureaucratic palace culture before alphabetic epic, clarifying what kind of world collapsed before Homer.
- **Limit:** Administrative writing and oral epic are different media and should not be collapsed.

## Comparison Objects

### 05. Trojan War amphora scene

- **Type:** artwork
- **Source:** https://commons.wikimedia.org/wiki/Category:Trojan_War_in_ancient_Greek_pottery
- **Rights:** external_link_only
- **What to notice:** Pottery scenes show how epic became shared public imagery, not only a private text.
- **Limit:** Category pages gather later visual traditions; each object needs separate dating for close claims.
