# Corpus

This folder holds public corpus annotations and now includes the first Geo-Strategy batch (`geo-01` through `geo-12`), the first Civilization spine (`civ-01` through `civ-60`), the Great Books Volume V spine (`gb-01` through `gb-09`), and the `ph-civ` derived study corpus. Within the Civilization spine, `civ-02` through `civ-60` are in review after initial commentary analysis.

The first public pilot is intentionally small and manual, and it should continue to grow in reviewed batches.

Use [chapter-manifest.yaml](../chapter-manifest.yaml) as the fast routing table for the current chapter pairs. The default load contract is one chapter at a time.

## Entry Requirements

Each item must:

- preserve a stable source ID such as `geo-01`, `vi-16`, or `es-13`
- include the original source URL and publication date when known
- use the annotation template in `docs/annotation-block.md`
- label source, transcript, annotation, and review status
- include timestamps for video-derived claims where practical
- avoid long transcript text until rights/review policy is settled
