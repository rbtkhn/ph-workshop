# Corpus

This folder will hold public corpus annotations after the standards-only bootstrap.

V1 intentionally contains no real source items, transcripts, or excerpts. The first future pilot should be a small manual Geo-Strategy batch.

## Entry Requirements

Each item must:

- preserve a stable source ID such as `geo-01`, `vi-16`, or `es-13`
- include the original source URL and publication date when known
- use the annotation template in `docs/annotation-block.md`
- label source, transcript, annotation, and review status
- include timestamps for video-derived claims where practical
- avoid long transcript text until rights/review policy is settled
