# Great Books Corpus

This corpus holds public source pointers for Predictive History Volume V: Great Books. It is active and in review for `gb-01` through `gb-10`.

The corpus entries are not substitutes for the source videos, source transcripts, or commentaries. They provide traceable routing from each source ID into the book source transcript, commentary, and civ-ph placement card.

## Current Contract

- Public source IDs use `gb-01` through `gb-10`.
- Transfer book-track IDs such as `gb-ch01` are preserved only as metadata.
- Source transcript bodies preserve the transferred lecture body exactly after normalization when transfer text exists; `gb-10` is a direct YouTube auto-caption import and remains marked `needs_fidelity_review`.
- Commentaries are Great Books-native and in review.
- civ-ph placement cards provide orientation and re-entry, not duplicate commentary.
